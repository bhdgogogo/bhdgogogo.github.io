js怎么实现字符串排序

vuePress-theme-reco Seb    2024

js怎么实现字符串排序

seb 2023/7/7 String

现在有一个需求：输入一个字符串，比如"dcba"，我们想让他按照字典序排列，也就是"abcd"，而字符串又没有sort方法，把字符串转为数组后sort也不起作用，那该怎么实现呢？❓

# localeCompare

其实字符串也是有比较方法的 👇

js提供了字符串的对比方法localeCompare()，该方法返回的是一个数字用来表示一个参考字符串和对比字符串是排序在前，在后或者相同（你可以理解为是sort比较的规则）。

该方法基本不会单独使用，大部分时间是配合字符串排序使用的，所以用它来实现我们的需求是非常合适的

# 使用

curString.localeCompare(targetString,locales,options)

各变量的含义 👇

  • originString: 被比较的字符串 —— 必选
  • targetString: 你要用originString跟谁比较，谁就是targetString —— 必选
  • locales: MDN上的解释是用来表示一种或多种语言或区域的一个符合 BCP 47 标准的字符串或一个字符串数组，我个人的理解是，指定一系列字符的顺序，提供给对比字符串的对比顺序，比如字典序 —— 可选
  • options: 配置对象 —— 可选

返回值是一个数字，你可以理解为sort里比较的规则

目前的主流浏览器都返回的是1、0、-1三个值，但是也有其他情况，所以不可以用绝对的值等于1、-1这种去判断返回的结果

  • 返回值大于0：说明当前字符串curString大于对比字符串targetString
  • 返回值小于0：说明当前字符串curString小于对比字符串targetString
  • 返回值等于0：说明当前字符串curString等于对比字符串targetString

测试一下

const s1 = "aaa";
const s2 = "bb";
const s3 = "aaa";

console.log("s1和s2比较: ", s1.localeCompare(s2));
console.log("s2和s1比较: ", s2.localeCompare(s1));
console.log("s1和s3比较: ", s1.localeCompare(s3));

而且值得我们注意的是，字符串的长度会影响比较的结果 👇

const s1 = "a";
const s2 = "aa";
const s3 = "aaa";
const s4 = "aaaa";

console.log("s1和s2比较: ", s1.localeCompare(s2));
console.log("s1和s3比较: ", s1.localeCompare(s3));
console.log("s1和s4比较: ", s1.localeCompare(s4));

因为不知道我们比较的字符串的长度，所以它会遍历整个字符串来比较，如果一个字符串遍历结束了，另一个字符串还没有遍历结束，则会用空字符串和剩下的字符串比较，而空字符串是最小的，所以，"a"要小于"aa"，也就是排序结果是 —— "a", "aa", "aaa", "aaaa"

# 实现需求

我们发现这是用于两个字符串之间的方法，跟我们的问题好像没有什么关系，但是我上边说过，localeCompare的比较规则和sort里的比较规则类似，所以我们是不是可以利用sort来解决呢？sort是数组的方法，所以我们要把字符串先变为数组，sort之后再转为字符串即可

let str = "dcba";

let res = str
  .split("")
  .sort((a, b) => {
    return a.localeCompare(b);
  })
  .join("");

console.log("排序后的str: ", res);
// 排序后的str:  abcd

☝️ 我是把逻辑写到了一行语句中，其实就是先把字符串先转换为数组，排序后，再把数组转为字符串的顺序

# 使用场景

除了可以实现给字符串中的字符排序之外，localeCompare方法常常用于我们给字符串数组排序，尤其是存储人名的字符串数组，我们使用localeCompare来实现按照人名排序的效果是非常方便的

let name = ["张三", "李四", "王五", "安十", "赵六", "黄七"];
let originName = [...name];

console.log("原来的name: ", originName);

const newName = name.sort((a, b) => {
  return a.localeCompare(b);
});

console.log("排序后的name: ", newName);
