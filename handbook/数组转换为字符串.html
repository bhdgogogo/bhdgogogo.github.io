数组和字符串之间的转换

seb 2023/5/12 ArrayString

📝
在一些算法题或者一些场景下，我们需要把数组转换字符串，同时，我们可能对转换后的字符串形式有一些需求，这篇文章就来记录一下在JavaScript中如何把数组和字符串之间相互转换的几种方法

# 数组转换为字符串

# + " "

JS是一个弱解释型语言，所以我们可以利用这一点，直接将数组与字符串连接

const arr = [1, 2, 3, 4];
const test = arr + "";

console.log(test); // "1,2,3,4"
console.log(typeof test); // string

在JS中，利用+号把其他数据类型与" "连接后，该类型将会转换为字符串

# toString()

利用Object自带的toString()方法，同时，Array默认已经重写了该方法，调用toString()会直接返回数组内元素的字符串表示，元素之间默认是用 “，”号隔开

const arr = [1, 2, 3, 4];
const test = arr.toString();

console.log(test); // "1,2,3,4"
console.log(typeof test); // string

# join()

利用Array.prototype.join()方法，可以返回数组元素的字符串表示，同时，join()方法接收一个参数，该参数可以指定分割数组中元素的方式

如果不想要上述两种方法结果中的“，”，则可以通过join()方法来实现

🌨 直接对二维数组使用join("")的结果和一维数组是有区别的

const arr = [1, 2, 3, 4];

console.log(arr.join()); // 1,2,3,4
console.log(arr.join("")); // 1234
console.log(arr.join(" ")); // 1 2 3 4
console.log(arr.join("-")); // 1-2-3-4

注意传参时，""和“ ”是有区别的，前一个是没有空格分割，最后的字符串结果是"1234"，后一个有一个空格，结果是"1 2 3 4"

# 字符串转化为数组

# split()

作用：用于把一个字符串分割成字符串数组

MDN上的描述:

The split() method takes a pattern and divides a String into an ordered list of substrings by searching for the pattern, puts these substrings into an array, and returns the array.

split()方法接受一个pattern，并通过该pattern将String划分为有序的子字符串列表，将这些子字符串放入数组中，并返回该数组。

语法

str.split(separator,howmany)

参数

separator参数：必需填。字符串或正则表达式，只有遇到separator，才会进行分割。 howmany参数：可选。该参数可指定返回的数组的最大长度。如果设置了该参数，返回的子串不会多于这个参数指定的数组。如果没有设置该参数，整个字符串都会被分割，不考虑它的长度。

测试

测试1：

var str = "a b ,cc d e";
console.log("str: ", str);
// str:  a b ,cc d e

var a = str.split(" ");
console.log("a: ", a);
// 遇到空格就作为切割 —— 也就是把空格作为","来分割字符 —— 来返回结果
// a:  [ 'a', 'b', ',cc', 'd', 'e' ]

var b = str.split("");
console.log("b: ", b);
// b: ['a', ' ', 'b', ' ', ',', 'c', 'c', ' ', 'd', ' ', 'e']
//不传任何切割标志时，默认切割每一个字符，包括空格

var c = str.split(" ", 2);
console.log("c: ", c);
// c: ["I", "lover"]
// 以空格切割，并返回前两个数组元素

测试2：

let test1 = ":2:3:4:5";

console.log(test1.split(":"));
//["2", "3", "4", "5"]

let test2 = "|a|b|c";

console.log(test2.split("|"));
//["", "a", "b", "c"]
// 如果第一个元素是要切割的元素，则返回的数组第一个元素是空字符

测试3：

let h1 = "hello".split("");
console.log(h1);
// ["h", "e", "l", "l", "o"],分割单词

let h2 = "hello".split("", 3);
console.log(h2);
//["h", "e", "l"]，取按pattern分割后的前三个元素

let h3 = "hello".split("", 7);
console.log(h3);
// [ 'h', 'e', 'l', 'l', 'o' ] 最多只有5个元素，不超过7，就全部输出
