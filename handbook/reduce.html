数组方法 —— reduce()

seb 2023/5/6 Array

# 概念

reduce 方法在数组的每个元素上执行用户提供的回调函数，即“reducer”。它传入对前一个元素进行计算的返回值。

返回结果是单个值。它是在数组的所有元素上运行 reducer 的结果。

# 具体实现

reducer 函数逐个遍历数组元素，遍历时，reducer 函数将当前数组值添加到上一步的结果中，直到没有更多元素要添加，然后返回最后的结果 —— 单个值

# 参数

两个参数

  • 回调函数
  • 初始值 —— （initialValue）

# 回调函数

回调函数的四个参数：

  • preVal：前一个回调函数返回的前一个值。开始时是初始值
  • currVal：数组的当前元素
  • currentIndex：当前索引，可选的 ——一般不用
  • arrayToTraverse：要遍历的数组。可选的 —— 一般不用

注意：这些回调函数的参数名字是可以自己设置的，参数的作用对应的是参数的位置

# 初始值

初始值参数是可选的，它是第一次调用回调时，将前一个值初始化的值。

1️⃣ 如果未指定初始值，则将前一个值初始化为初始值，将当前值初始化为数组中的第二个值（相当于少遍历一次）

相当于从index为1的元素开始计算，因为index0被设置为了初始值， 如果数组长度为5，则在不设定初始值initialValue的情况下，reduce遍历4次

2️⃣ 如果指定了初始值，则从index0开始计算，如果数组长度为5，则在设定了初始值之后，reduce遍历5次

  1. reduce必须要有return，这样prev 才可以获取到上一次执行的结果，prev除了第一次能够拿到初始值，后边拿到的值都是undefined
  2. 空数组执行reduce操作且不提供初始值时，会报错，但设置了初始值的话，就不会报错了

没有return

let arr = [1, 2, 3, 4];

let sum = arr.reduce((prev, cur, index, arr) => {
  console.log("prev: ", prev);
  console.log("cur: ", cur);
  console.log("index: ", index);
  // return prev + cur;
});

/**
  prev:  1
  cur:  2
  index:  1
  prev:  undefined
  cur:  3
  index:  2
  prev:  undefined
  cur:  4
  index:  3
 */

空数组没有设置初始值

let arr = [];

let sum = arr.reduce((prev, cur, index, arr) => {
  console.log(prev, cur, index);
  return prev + cur;
})

// 报错：TypeError: Reduce of empty array with no initial value

空数组设置了初始值

let arr = [];

let sum = arr.reduce((prev, cur, index, arr) => {
  console.log(prev, cur, index);	// 没有输出结果，因为是空数组
  return prev + cur;
}, 0);

console.log("arr: ", arr); // arr:  []
console.log("sum: ", sum); // sum:  0

# 场景分析

# 对数组元素求和

最常用的场景


const prices = [5.99, 2.99, 3.99, 11.59]
const sum = prices.reduce((preVal, currVal) => preVal + currVal, 0);
// sum:  24.560000000000002

# 展平数组

将多维数组转换为一维数组

var rows = [
  [2, 3, 5],
  [1, 2, 4],
  [8, 5, 5],
];
console.log("rows: ", rows);
// rows:  [ [ 2, 3, 5 ], [ 1, 2, 4 ], [ 8, 5, 5 ] ]
var matrixElements = rows.reduce(function (prev, current) {
  return prev.concat(current);
});

console.log("matrixElements: ", matrixElements);

// matrixElements: [2, 3, 5, 1, 2,4, 8, 5, 5]

# 从数组中获取对象

可以从字符串数组中，根据你的要求，得到一个对象并返回 —— 数组转为对象

注意这里初始值要传入一个对象

const apples = ['green', 'red', 'red', 'yellow', 'red', 'yellow', 'green', 'green'];
var appleMap = apples.reduce((prev, apple) => {
  if (prev[apple] >= 1) prev[apple]++;
  else prev[apple] = 1;
  return prev;
}, {});

console.log("appleMap: ", appleMap);
// appleMap:  { green: 3, red: 3, yellow: 2 }

# 求给定数组的最大子数组和

从给定的数组中，找到它的一个最大子数组和，并返回该最大子数组和

注意：子数组就说明了是连续的

const max = (nums, sum = 0) => {
  return nums.reduce((max, cur) => {
    sum = Math.max(sum + cur, cur);
    return Math.max(sum, max);
  }, nums[0]);
};

console.log("max: ", max([-1, -2, -3, -2, -5, -1, -3, -9, -1, -6])); // max: 8
console.log("max: ", max([1, -2, 3, -2, 5, -1, 3, -9, 1, 6])); // max: 8

# reduceRight()

原理与reduce相同，只是与reduce的执行顺序相反，它是从后往前执行的

const prices = [5.99, 2.99, 3.99, 11.59];

let sum = prices.reduceRight(function (preVal, currVal) {
  console.log("prev: ", preVal, "curr: ", currVal);
  return preVal + currVal;
});
console.log(sum);

// prev:  11.59 curr:  3.99
// prev:  15.58 curr:  2.99
// prev:  18.57 curr:  5.99
// 24.560000000000002
