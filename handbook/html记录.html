# a标签

属性如下：

  • href属性表示你先把通过点击a标签跳转到哪个位置
  • target表示在哪里打开跳转的内容，默认是_self，表示在当前窗口内跳转
  • target设置为_blank，表示在新窗口跳转

# 自定义列表

  • dl表示自定义列表的整体，用于包裹dt和dd
  • dt表示自定义列表的主题 —— 相当于表头
  • dd表示列表的每一项内容 —— 相当于ul的li中的内容

请写出列表项为"nowcoder"且列表项内容也为"nowcoder"的自定义列表。

# 语义化标签

  • header表示头部标签
  • nav表示导航标签

请使用语义化标签创建头部标签且包含导航标签，有且仅有一个头部标签和一个导航标签。

# 音频标签audio

属性如下：

  • src表示媒体文件路径
  • controls表示在网页内显示音频控件
  • autoplay表示打开网页时自动播放
  • loop表示循环播放

请写出具有控件功能的音频媒体标签。

# 视频标签video

属性如下：

  • controls显示控制组件
  • loop表示循环播放
  • muted表示静音播放
  • preload表示页面加载再播放
  • width/height表示视频的宽高
  • autoplay表示自动播放

请写出具有当视频的媒体数据加载期间发生错误时执行某个方法事件的视频媒体标签
