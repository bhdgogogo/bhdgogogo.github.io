js删除字符串中指定字符

seb 2023/6/30 String

在前端日常开发中，经常会处理字符串，这篇文章就来总结一下删除字符串中指定字符的几种方法

# replace()

MDN 上的描述如下：

replace() 方法返回一个由替换值（replacement）替换部分或所有的模式（pattern）匹配项后的新字符串。模式可以是一个字符串或者一个正则表达式，替换值可以是一个字符串或者一个每次匹配都要调用的回调函数。如果 pattern 是字符串，则仅替换第一个匹配项

不会改变原字符串

# 参数

  • 第一个参数指定要替换的字符串或正则表达式
  • 第二个参数指定用于替换的字符串

# 使用

  1. 用空字符串替换指定元素 —— 实现删除

只会删除第一个指定的元素！

let str = "my heart never lie";
str = str.replace("e", "");
console.log("str: ", str);

// str:  my hart never lie
  1. 用正则的方式来设置指定元素 —— 比如删除字符"e"

因为是正则匹配，会删除所有指定的元素！

let str = "my heart never lie";
str = str.replace(/e/g, "");
console.log("str: ", str);
// str:  my hart nvr li
  1. 使用函数来删除指定元素

replace()的第二个参数可以设置为函数，通过函数来操作

函数接收四个参数

  • match: 匹配的子串
  • p1,p2,...假如 replace() 方法的第一个参数是一个RegExp 对象，则代表第 n 个括号匹配的字符串。（对应于上述的$1，$2 等。）例如，如果是用/(\a+)(\b+)/ 这个来匹配，p1 就是匹配的 \a+，p2 就是匹配的 \b+。
  • offset: 匹配到的子字符串在原字符串中的偏移量。（比如，如果原字符串是 'abcd'，匹配到的子字符串是 'bc'，那么这个参数将会是 1）
  • string: 原字符串

在实际实现中可以直接通过 return 来返回替换结果

只会删除第一个指定元素，后面的不会删除

let str = "my heart never lie";

const replacer = function (match, offset, string) {
  console.log("match: ", match);
  console.log("offset: ", offset);
  console.log("string: ", string);
  return "";
};

str = str.replace("e", replacer);

console.log("str: ", str);

/* 
  match:  e
  offset:  4
  string:  my heart never lie
  str:  my hart never lie
*/

# split 配合 join

  • 先利用split将字符串分割子串数组，分割时可以指定分割符 —— 利用分割符来实现删除的效果
  • 再将数组转为字符串 —— join

会替换所有的指定字符（分割符）

实现 👇

let str = "my heart never lie";

let strArr = str.split("e");
console.log("strArr: ", strArr);
str = strArr.join("");
console.log("str: ", str);

// strArr:  [ 'my h', 'art n', 'v', 'r li', '' ]
// str:  my hart nvr li

# slice 和 substring

这两个方法都是利用取子串的方式 —— 以删除字符元素的下标为中心，分成左右两部分的子串，然后再将两个子串合并，来实现删除的效果

这种方法只能用于已知字符串的内容的情况，如果是对于输入的内容，则不能用该方法来实现

let str = "my heart never lie";

let newStr1 = str.slice(0, 4) + str.slice(5);
let newStr2 = str.substring(0, 4) + str.substring(5);

console.log("newStr1: ", newStr1);
console.log("newStr2: ", newStr2);

// newStr1:  my hart never lie
// newStr2:  my h、art never lie

# 总结

通过分析，发现JS中有很多方法可以实现删除指定字符的需求，我们需要根据我们的场景，来选择适用的方法
